Бот для Телеграмм

Вы можете играть с вашим другом прямо в Телеграм-чате @BiddingTicTacToeBot.

Правила игры

  • Каждому игроку выдаётся по 100 жетонов на начало игры.
  • Чтобы выиграть право хода игроки делают ставки.
  • Ход делает тот кто сделал большую савку. Если ставки одинаковы выигрывает игрок с большим балансом. Если ставки и балансы одинаковы выигрывает тот кто сделал ставку первым.
  • Сделавший ход передаёт свою ставку другому игроку.
  • Игру выигрывает тот кто построит непрерывную линию от края до края или из угла в угол.

Исходные коды открыты!

This game is written in Go language using Strongo Bots Framework.
